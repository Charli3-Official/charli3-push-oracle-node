# Getting Support

Thank you for using Charli3. Here is how to get help depending on your situation.

## Documentation

The primary resource for integration guides, architecture explanations, and
API references is the official documentation site:

📖 **[docs.charli3.io](https://docs.charli3.io)**

## Community Help

For questions, ideas, and general discussion:

- **Discord** — [discord.gg/Tnsc3HCA3A](https://discord.gg/Tnsc3HCA3A)

## Bug Reports

If you have found a reproducible bug, please open an issue in the relevant
repository using the **Bug Report** template.

## Security Vulnerabilities

**Do not open a public issue for security vulnerabilities.**  
Please follow the responsible disclosure process described in
[SECURITY.md](./SECURITY.md) and email **info@charli3.io** directly.

## Feature Requests

Feature requests are welcome via an issue using the
**Feature Request** template in the relevant repository.
